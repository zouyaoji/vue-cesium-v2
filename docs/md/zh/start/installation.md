# 安装

## NPM

```bash
npm install vue-cesium@2.4.2 --save
```

## YARN

```bash
yarn add vue-cesium@2.4.2
```

## CDN

```js
<!-- include Vue -->
<script src="https://unpkg.com/vue@2.6.12/dist/vue.min.js"></script>
<!-- include VueCesium -->
<script src="https://unpkg.com/vue-cesium@2.4.2/lib/index.umd.js"></script>
```
